HT Catalog bookmarklets

For any of these that you're interested in, you should be able to just drag the appropriate link to your bookmark bar.

  • HT2M: Find the current record in Mirlyn. Only works from the Single Record page.
  • HT2MC, a new HT2MC that uses the first HT Volume ID to search Mirlyn Classic for the equivalent record. The old version used the Record ID, which isn't always the same between the two systems. Note that the MARC in Mirlyn Classic may not be the same as the MARC used by the HathiTrust!.
  • HT Marc: Show a text representation of the MARC record behind the current page. Only works from a single record page.
    • HT_JSON, for a marc-in-json view of the MARC. Most useful if you're running something like JSONView.
    • HT_XML, for a MARC-XML view of the MARC. Most useful if you're running something like XMLTree.
  • HTID_List: Redirect to a list of HT volume IDs (one per line) for all the volumes attached to records in the current search. Only works from the Search Results page.
