HT Catalog bookmarklets

For any of these that you're interested in, you should be able to just drag the appropriate link to your bookmark bar.

HathiTrust MARC views

Bookmarklet Context Description
HT Marc Record page A human-readable view of the MARC underlying the current record. Note: I have complete control over this. If changes to its layout or formatting would be useful, just let me know.
HT_XML Record page Gives the MARC-XML for the current record

Getting lists of HT IDs

Bookmarklet Context Description
HTID_List Search Results Redirect to a list of HT volume IDs (one per line) for all the volumes attached to records in the current search. Might take a while for large result sets!

Hathitrust to Mirlyn

Please Note: The MARC underlying Mirlyn/Mirlyn Classic is not the same as that underlying a HathiTrust record.

Bookmarklet Context Description
HT2M Record page Go to the equivalent record in Mirlyn
HT2MC Record page Go to the equivalent record in Mirlyn Classic
